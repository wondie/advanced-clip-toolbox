Batch Clip Raser By Shapefiles

Title Batch Clip Raser By Shapefiles

Summary

The tool enables you to clip a single raster file into multipple rasters using shapefiles in a folder. The output rasters will combine the shapefile name and the raster file name and they are saved in the output directory.


Usage

The tool is useful when you have large raster data and you want to clip it into multipple smaller files using a shapefile.

Syntax

BatchClipRaserByShapefiles (Raster_File, Shapefiles_Folder, Output_Folder)

Parameter Explanation Data Type
Raster_File

The raster file to be clipped.

 Raster Dataset
Shapefiles_Folder

A folder containing shapefiles.

 Workspace
Output_Folder

The output folder where the clipped rasters will be saved.
